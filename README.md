# This is a template server for the ZIPBOLT project

## Setup

- Clone the repository
- code uv add requirements.txt
- Install the dependencies
- Run the server

## Dependencies

- Flask
- SQLAlchemy
- Python 3.12/n

## Run the server

- python -m uvicorn main:app --reload
- url: [http://localhost:8000](http://localhost:8000)
